Skenario 3: Fungsi Rekursif
User: "Apa itu fungsi rekursif dan bagaimana cara menggunakannya di Python?"
AI: "Fungsi rekursif adalah fungsi yang memanggil dirinya sendiri. Contohnya, menghitung faktorial dari sebuah angka:

python
Copy code
def faktorial(n):
    if n == 1:
        return 1
    else:
        return n * faktorial(n - 1)

print(faktorial(5))  # Output: 120
Fungsi faktorial memanggil dirinya sendiri untuk menghitung faktorial dari angka yang diberikan."

Skenario 7: Exception Handling dengan else dan finally
User: "Bagaimana cara menggunakan blok else dan finally dalam penanganan error di Python?"
AI: "Blok else dijalankan jika tidak ada exception yang ditangkap, dan blok finally dijalankan selalu, baik ada exception maupun tidak. Contohnya:

python
Copy code
try:
    angka = int(input("Masukkan sebuah angka: "))
    hasil = 10 / angka
except ZeroDivisionError:
    print("Error: Tidak bisa membagi angka dengan nol.")
else:
    print(f"Hasil: {hasil}")
finally:
    print("Blok finally dijalankan selalu.")
Blok else akan dijalankan jika tidak ada exception, sedangkan blok finally akan dijalankan selalu."

Skenario 9: Menggunakan map untuk Transformasi List
User: "Bagaimana cara menggunakan map untuk transformasi list di Python?"
AI: "Fungsi map digunakan untuk menerapkan fungsi tertentu ke setiap elemen dalam list. Contohnya:

python
Copy code

Skenario 10: Menggunakan filter untuk Menyaring List
User: "Bagaimana cara menggunakan filter untuk menyaring list di Python?"
AI: "Fungsi filter digunakan untuk menyaring elemen dalam list berdasarkan kondisi tertentu. Contohnya:

python
Copy code
