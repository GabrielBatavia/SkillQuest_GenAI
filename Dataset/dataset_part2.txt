Skenario 2: Variabel dan Tipe Data
User: "Apa saja tipe data dasar di Python?"
AI: "Di Python, tipe data dasar meliputi:

int: Angka bulat, contoh: a = 10
float: Angka desimal, contoh: b = 10.5
str: Teks atau string, contoh: c = "Hello"
bool: Nilai boolean, contoh: d = True
list: Daftar yang dapat diubah, contoh: e = [1, 2, 3]
tuple: Daftar yang tidak dapat diubah, contoh: f = (1, 2, 3)
dict: Koleksi pasangan kunci-nilai, contoh: g = {"key": "value"}"
Skenario 3: Pengulangan dan Kondisi
User: "Bagaimana cara membuat loop for di Python?"
AI: "Untuk membuat loop for di Python, Anda dapat menggunakan sintaks berikut:

python
Copy code
for i in range(5):
    print(i)
Kode di atas akan mencetak angka 0 hingga 4. Fungsi range(5) menghasilkan urutan angka dari 0 hingga 4, dan loop for akan mengiterasi melalui urutan tersebut, mencetak setiap angkanya."

Skenario 4: Fungsi
User: "Bisakah kamu menjelaskan bagaimana cara mendefinisikan fungsi di Python?"
AI: "Tentu! Untuk mendefinisikan fungsi di Python, gunakan kata kunci def diikuti dengan nama fungsi dan tanda kurung. Contohnya:

python
Copy code
def greet(name):
    return f"Hello, {name}!"
Fungsi greet ini menerima satu argumen name dan mengembalikan string sapaan. Anda bisa memanggil fungsi ini dengan greet("Alice") untuk mendapatkan hasil "Hello, Alice!"."

sh
Copy code
pip install requests
Setelah instalasi selesai, Anda bisa mengimpor dan menggunakan library tersebut dalam kode Anda:

python

Skenario 6: Error Handling
User: "Bagaimana cara menangani error di Python?"
AI: "Di Python, Anda bisa menggunakan blok try dan except untuk menangani error. Contohnya:

python
Copy code
try:
    result = 10 / 0
except ZeroDivisionError:
    print("Tidak bisa membagi angka dengan nol.")
Kode di atas akan menangkap error ZeroDivisionError dan mencetak pesan 'Tidak bisa membagi angka dengan nol.' alih-alih menghentikan eksekusi program."

Skenario 7: Mengelola File
User: "Bagaimana cara membaca dan menulis file di Python?"
AI: "Untuk membaca dan menulis file di Python, Anda bisa menggunakan fungsi bawaan open. Contohnya, untuk menulis ke file:

python
Copy code
with open('example.txt', 'w') as file:
    file.write('Hello, world!')
Dan untuk membaca file:

python
Copy code
with open('example.txt', 'r') as file:
    content = file.read()
    print(content)
Menggunakan with memastikan bahwa file ditutup secara otomatis setelah operasi selesai."
