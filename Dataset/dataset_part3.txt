Kode di atas akan membaca file data.csv dan mencetak lima baris pertama dari data tersebut."

Skenario 2: Membersihkan Data
User: "Bagaimana cara menangani data yang hilang di Python?"
AI: "Dengan pandas, Anda bisa menggunakan metode dropna atau fillna untuk menangani data yang hilang. Contohnya:

python
Copy code
import pandas as pd

data = pd.read_csv('data.csv')
data_cleaned = data.dropna()  # Menghapus baris dengan data yang hilang
# Atau
data_filled = data.fillna(0)  # Mengisi data yang hilang dengan nilai 0
print(data_cleaned.head())
print(data_filled.head())
Metode dropna akan menghapus baris yang mengandung data yang hilang, sedangkan fillna akan mengisi data yang hilang dengan nilai yang Anda tentukan."

Skenario 3: Visualisasi Data
User: "Bagaimana cara membuat grafik di Python?"
AI: "Anda bisa menggunakan library matplotlib untuk membuat grafik. Contohnya:

python
Copy code
import matplotlib.pyplot as plt

data = [1, 2, 3, 4, 5]
plt.plot(data)
plt.title('Contoh Grafik')
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.show()
Kode di atas akan membuat grafik garis sederhana dengan data yang diberikan."
